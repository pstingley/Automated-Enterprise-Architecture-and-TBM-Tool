Automated Enterprise Architecture and TBM Migration Toolset

The Python programs herein were written to be extremely simple for those who don't know Python.  
The_Mill.py cleans, sorts and counts the applications identified by any inventory tool capable of providing a product's name followed by its version.  It was created because enterprises often have more than 1M product/versions on their network.  This cuts the data down
as it sorts and counts it, so it will fit in Excel

The python program is named The_Mill.py because a user simply runs the data through the mill and out comes a data file that they can paste into the provided spreadsheet.

It only takes a few minutes to run the program and paste it into the spreadsheet.

inventory_vulnerability_assessor.py takes an .xlsx, .XLS or CSV file with Product, version as its column names and does a lookup
to find out about any known vunlerabilities with the software listed.
